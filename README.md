# Dynamic_Feature_B4E

This project is used to train the model with different B4E dataset options.

## Run

Choose the dataset you want to use by passing the `--ds` argument.

### Run with `B4E_bal`

```bash
python trainModel.py --ds B4E_bal
````

### Run with `B4E`

```bash
python trainModel.py --ds B4E
```

## Notes

* `B4E_bal` uses the balanced dataset folder.
* `B4E` uses the original dataset folder.
